# 💒 அபிராமி & கௌரிசங்கர் - Wedding Invitation

## How to use in VS Code

1. Open this folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**

## ➕ How to add the couple's photo

1. Save your couple's photo as **`couple.jpg`** in this same folder
2. Refresh the browser — the photo will automatically appear in the oval frame!
   - The photo works best at **portrait orientation** (taller than wide)
   - Ideal size: around 400×500 px

## 📱 Payment Buttons (UPI)

- **Google Pay** and **PhonePe** buttons are set to UPI ID: `9840070498@ybl`
- On **Android phones**, clicking will directly open the respective payment app
- On **iPhone / Desktop**, it opens the generic UPI link
